How to change app icons on macOS Sequoia (and undo it)

macOS Customization

Steps (built-in, the trustworthy way)

  1. In Finder, go to Applications and Right-click → Get Info on the app.
  2. Open your replacement image (PNG/ICNS) in Preview, press ⌘A then ⌘C to copy.
  3. In the app’s Get Info panel, click the tiny icon at the top-left so it’s highlighted, then press ⌘V to paste.

Undo / reset the icon

  • Select the tiny top-left icon in Get Info, press Delete to remove the custom icon. If the Dock shows the old icon, you may need to rebuild the icon cache.

Limits you should know

  • Apple’s System Integrity Protection (SIP) can block changes to some built-in system apps. Disabling SIP is not recommended for most users.
